# SRS - Food Ordering App

## Introduction
Dokumen ini berisi kebutuhan sistem aplikasi pemesanan makanan berbasis web.

## System Overview
Sistem memungkinkan pengguna untuk:
- Login dan registrasi
- Melihat menu makanan
- Melakukan pemesanan
- Melakukan pembayaran

## User Roles
- Customer
- Admin

## Functional Requirements
FR-01: User dapat login
FR-02: User dapat melihat menu
FR-03: User dapat melakukan pemesanan

## Non-Functional Requirements
NFR-01: Sistem cepat
NFR-02: Sistem aman
