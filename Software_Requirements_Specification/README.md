# Software Requirements Specification (SRS)

## Deskripsi
Dokumen ini berisi kebutuhan sistem dari aplikasi login dan registrasi berbasis web.

## Kebutuhan Fungsional
- User dapat melakukan registrasi
- User dapat login
- Sistem menampilkan verifikasi
- Sistem menyimpan data user
- User dapat mengakses dashboard

## Kebutuhan Non-Fungsional
- Sistem mudah digunakan
- Tampilan sederhana dan responsif
- Menggunakan localStorage (tanpa database)

## Tujuan
Sebagai dasar dalam pengembangan sistem.
