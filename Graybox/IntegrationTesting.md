# Integration Testing

## Tujuan
Menguji integrasi antar modul dalam Sistem Informasi Perpustakaan.

| No | Skenario Pengujian | Hasil |
|----|-------------------|----------------------|
| 1 | Login menuju Dashboard | Berhasil |
| 2 | Dashboard menuju Peminjaman Buku | Berhasil |
| 3 | Peminjaman Buku ke Database | Data tersimpan |
| 4 | Pengembalian Buku ke Database | Data diperbarui |
| 5 | Logout kembali ke Login | Berhasil |

## Kesimpulan
Seluruh modul dapat terhubung dan berinteraksi dengan baik.
