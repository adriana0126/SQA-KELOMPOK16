# Security Testing

## Tujuan
Menguji keamanan dasar Sistem Informasi Perpustakaan.

| No | Skenario Pengujian | Hasil Yang Diharapkan |
|----|-------------------|----------------------|
| 1 | SQL Injection pada Login | Ditolak sistem |
| 2 | Input Script HTML pada Register | Tidak dieksekusi |
| 3 | Akses admin tanpa login | Ditolak |
| 4 | Manipulasi URL | Tidak dapat mengakses data lain |
| 5 | Input karakter khusus | Sistem tetap berjalan normal |

## Kesimpulan
Sistem memiliki mekanisme keamanan dasar yang berfungsi dengan baik.
