# Session Testing

## Tujuan
Menguji pengelolaan session pada Sistem Informasi Perpustakaan.

| No | Skenario Pengujian | Hasil Yang Diharapkan |
|----|-------------------|----------------------|
| 1 | Login dengan akun valid | Session terbentuk |
| 2 | Logout | Session terhapus |
| 3 | Akses dashboard tanpa login | Ditolak sistem |
| 4 | Session timeout | User logout otomatis |
| 5 | Session aktif | Dashboard dapat diakses |

## Kesimpulan
Session pada Sistem Informasi Perpustakaan berjalan sesuai fungsi yang diharapkan.
