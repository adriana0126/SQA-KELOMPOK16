## Tujuan
Menguji validasi input pada Sistem Informasi Perpustakaan.

| No | Skenario Pengujian | Hasil |
|----|-------------------|--------|
| 1 | Nama pengguna kosong saat registrasi | Ditolak |

| 3 | Password kurang dari 8 karakter | Ditolak |
