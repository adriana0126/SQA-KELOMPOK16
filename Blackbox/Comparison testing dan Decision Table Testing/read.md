# Comparison Testing dan Decision Table Testing

## Tujuan
Menguji kombinasi input username dan password pada login.

| Username | Password | Hasil |
|-----------|-----------|--------|
| Benar | Benar | Login Berhasil |
| Benar | Salah | Login Gagal |
| Salah | Benar | Login Gagal |
| Salah | Salah | Login Gagal |
| Kosong | Kosong | Validasi Ditampilkan |

## Kesimpulan
Sistem dapat membedakan kombinasi input yang valid dan tidak valid.
