# Endurance Testing dan Cause Effect Testing

## Tujuan
Menguji hubungan sebab-akibat pada fitur peminjaman buku.

| No | Cause (Penyebab) | Effect (Akibat) |
|----|------------------|-----------------|
| 1 | Buku tersedia | Peminjaman berhasil |
| 2 | Buku habis | Peminjaman ditolak |
| 3 | User sudah login | Dapat meminjam |
| 4 | User belum login | Tidak dapat meminjam |
| 5 | Data peminjaman lengkap | Data tersimpan |

## Kesimpulan
Hubungan antara kondisi input dan hasil sistem berjalan dengan baik.
